Run the full local rapp-zoo

This static page is a thin slice — Discover + Inspect only, no local-collection / drag-drop-import / lay-egg / summon / hatch / start-stop. For the full feature surface, install the local Flask-backed rapp-zoo. It runs at localhost:7070 and adds:

  • 📁 My collection — every organism on this device, grouped by rappid
  • 📥 Drag-drop import — drop a .egg on the page, the zoo saves and registers it
  • 🥚 Lay egg — pack any organism's repo into a fresh portable .egg
  • 🐣 Summon / hatch — materialize an egg into ~/.rapp/twins/<rappid>/
  • ▶️ Start / stop / reveal-in-finder — process control for every organism

Install with one curl:

curl -fsSL https://kody-w.github.io/rapp-zoo/installer/install.sh | bash && \
bash ~/.rapp-zoo/installer/start.sh

Then open http://127.0.0.1:7070.
